# Booking Vehicle

## Overview

Booking Vehicle App is a web application that allows users to browse and book vehicles. The platform provides a catalog of available vehicles. Users can also add vehicles to their favorites list. Additionally, the project incorporates an intuitive and attractive design.

## Features

- **Vehicle Browsing:**
  Browse a catalog of available vehicle.
- **Favorites:**
  Add vehicle to your favorites list for quick access.
- **Booking Information:**
  View detailed information about each vehicle.
- **User-Friendly Interface:**
  Enjoy a seamless experience with an intuitive and attractive design.

## Tech Stack

- **Frontend:**
  - React: A popular JavaScript library for building user interfaces.
  - Styled Components: For styling and theming components in a dynamic and maintainable way.
  - Redux Toolkit: State management library for handling application-level state.
  - React Router: Declarative routing in single-page applications.
- **Backend:**
  - MockAPI: A tool for creating fake APIs, used for simulating backend functionality during development.
- **Deployment:**
  - GitHub Pages: Hosting platform for deploying the frontend.
- **Version Control:**
  - Git: Distributed version control system for tracking changes in the codebase.
- **Development Tools:**
  - Visual Studio Code: An integrated development environment for efficient coding.
  - GitHub: Version control platform for collaborative development.

## Getting Started

These instructions will help you set up and run the Booking Vehicle app on your local machine for development and testing purposes.

### Prerequisites

- Node.js and npm installed

### Installation

1.  Clone the repository:

    ```bash
    git clone https://github.com/ILIANASEMENOVA/test-booking-vehicle.git

    cd test-booking-vehicle
    ```

2.  Install dependencies:

    ```bash
    npm install
    ```

### Run the application:

1. Run app using following command:

   ```bash
   npm run dev
   ```
